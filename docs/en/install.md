# Installation

## Requirements
- **Paper** server (recommended) 1.21+
- Java **21**

## Steps
1. Put the plugin jar into `plugins/`.
2. Start/restart the server.
3. Edit configs in `plugins/FunTalismans/`.

## Optional integrations
- ProtocolLib, PlaceholderAPI, WorldEdit/WorldGuard, NBT-API (optional hooks)
