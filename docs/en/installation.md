# Installation

## What is FunTalismans?
FunTalismans is a plugin that lets you create configurable **talismans** using `.conf` (HOCON) files.

## Requirements
- Paper (recommended)
- Java 21+
- Optional hooks: ProtocolLib, PlaceholderAPI, WorldGuard/WorldEdit, etc.

## Install
1. Download the `.jar`.
2. Put it into `plugins/`.
3. Start the server once.
4. Configure talismans in the plugin folder.
5. Reload to apply changes (see “Reload”).
