# Instalación

## Requisitos
- Servidor **Paper** (recomendado) para 1.21+
- Java **21**

## Pasos
1. Descarga el `.jar` de FunTalismans.
2. Colócalo en la carpeta `plugins/`.
3. Reinicia el servidor (o inicia por primera vez).
4. Se crearán carpetas/configs de FunTalismans.

## Dependencias / Integraciones (opcionales)
FunTalismans puede integrarse con:
- **ProtocolLib**
- **PlaceholderAPI**
- **WorldEdit / WorldGuard**
- **NBT-API** (según tu instalación)

Si no las usas, el plugin igual funciona (son “hooks” opcionales).

## Primer arranque
Al primer arranque revisa:
- `plugins/FunTalismans/config.conf`
- `plugins/FunTalismans/rarities/` (rareza)
- `plugins/FunTalismans/talismans/` (tus talismanes)

> Si el plugin no genera archivos, revisa consola por errores de permisos o rutas.
