# Crear un talismán

1) Crea un archivo `.conf` en la carpeta de talismanes.
2) Define la sección del talismán.

```hocon
vampire {
  id = "vampire"
  name = "&cVampire Talisman"
  material = "NETHER_STAR"
  rarityId = "epic"
  glow = true
  unbreakable = true
  lore = [
    "&7Absorbe vida con cada golpe."
  ]
}
```

3) Recarga el plugin (ver “Recarga”).
