# Comandos y permisos

> Esta sección depende de cómo tengas tus comandos registrados.
> Si quieres, puedo completar esta página con los comandos reales si pegas aquí tu `plugin.yml` o `paper-plugin.yml`.

Recomendación típica:
- `/funtalismans reload` — recarga configs/talismanes
- `/funtalismans give <player> <id> [amount]` — dar talismanes
- `/funtalismans list` — lista ids

Permisos típicos:
- `funtalismans.admin`
- `funtalismans.reload`
- `funtalismans.give`
- `funtalismans.use`
