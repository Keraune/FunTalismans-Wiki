# Instalación

## ¿Qué es FunTalismans?
FunTalismans es un plugin que permite crear **talismanes** configurables mediante archivos `.conf` (HOCON).

## Requisitos
- Paper (recomendado)
- Java 21+
- Hooks opcionales: ProtocolLib, PlaceholderAPI, WorldGuard/WorldEdit, etc.

## Instalación
1. Descarga el `.jar`.
2. Colócalo en `plugins/`.
3. Inicia el servidor una vez.
4. Configura tus talismanes en la carpeta del plugin.
5. Recarga para aplicar cambios (ver “Recarga”).
